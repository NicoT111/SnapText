# 📄 SnapText

**SnapText** ist eine schnelle und moderne macOS-Menüleisten-App, mit der sich Text aus Bildern und Bildschirmbereichen automatisch erkennen und kopieren lässt.

## ✨ Funktionen

* 🖼️ Texterkennung (OCR) aus Bildern und Screenshots
* 📋 Automatisches Kopieren in die Zwischenablage
* 📚 Integrierter Verlauf erkannter Texte
* ⌨️ Globale Tastenkombination
* 🔔 Optionale macOS-Benachrichtigungen
* ⚡ Schnelle und vollständig lokale Verarbeitung
* 🎯 Modernes Design im Apple-Stil

## 🚀 So funktioniert es

1. SnapText starten.
2. Die festgelegte Tastenkombination drücken.
3. Einen Bereich auf dem Bildschirm markieren.
4. Der Screenshot wird automatisch erstellt.
5. Der enthaltene Text wird erkannt.
6. Das Ergebnis wird direkt in die Zwischenablage kopiert und optional im Verlauf gespeichert.

## 💻 Voraussetzungen

* macOS 13 oder neuer
* Bildschirmaufnahme-Berechtigung
* Mitteilungsberechtigung (optional)

## 🔒 Datenschutz & Sicherheit

SnapText verarbeitet alle Daten ausschließlich lokal auf deinem Mac.

* ✅ Keine Cloud
* ✅ Kein Tracking
* ✅ Keine Datensammlung
* ✅ Keine Werbung
* ✅ Sicher und vollständig lokal
